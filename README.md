# 🌤 AirPeek

🌍 AirPeek, kullanıcıların şehir arayarak güncel hava durumunu görüntüleyebildiği modern bir web hava durumu uygulamasıdır. 
Uygulama, gerçek zamanlı hava durumu verilerini bir Weather API üzerinden çekerek kullanıcıya sıcaklık, nem, rüzgar ve UV gibi bilgileri sunar. 
Modern glassmorphism tasarım, dinamik arka planlar ve interaktif şehir kartları ile kullanıcı deneyimi ön planda tutulmuştur.

### 🌐 Özellikler

#### 🌍 Hava Durumu Sistemi

- Gerçek zamanlı hava durumu verisi
- Sıcaklık görüntüleme
- Hissedilen sıcaklık
- Nem oranı

#### 🎨 Arayüz Özellikleri

- Animasyonlu şehir kartları

### 🧠 Bu Projede Neler Öğrendim?

Bu proje ile API kullanımı ve veri çekme mantığını öğrendim. Projede özellikle şu konularda deneyim kazandım:

- API üzerinden veri çekme
- Axios ile HTTP istekleri
- JSON veri işleme
- React state yönetimi
- API verisini arayüze bağlama
- Dinamik UI güncellemeleri

Gerçek bir API kullanarak frontend uygulamalarında veri yönetimi pratiği kazandım.
